Abstract

The $K_{2,s+1}$-free graphs $F(q^t,s)$ defined by F\"uredi were investigated recently by Livinsky when $s=\frac{q^{t}-1}{q-1}$. In this note we determine the exact relationship of these graph to the $K_{t+1, t!+1}$-free projective normgraphs $NG(q,t)$ and find that they are the same up to a few exceptional vertices. We prove this via defining a $K_{t+1, t!+1}$-free generalization of the Erd\H os-R\'enyi polarity graph, which contains both as an induced subgraph.
